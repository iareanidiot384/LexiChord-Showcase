# LexiChord

Narrative-to-music prototyping platform for transforming scene text into playable harmonic ideas.

## Overview
LexiChord helps creators move from words to sound in minutes. It interprets narrative tone, maps it to musical intent, and produces harmonically coherent progressions with real-time audition and export-ready output.

This project demonstrates my ability to design and ship an end-to-end creative tool that combines:
- Interactive product thinking for non-technical users
- Structured sentiment-to-audio decision pipelines
- Deterministic music generation workflows
- Fast feedback loops in a browser-first UX

## What It Enables
- Convert scene text into emotional coordinates and musical direction
- Generate context-aware chord progressions for scoring ideation
